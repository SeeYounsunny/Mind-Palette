# Mind Palette - 5분 발표 자료 구조

## 슬라이드 1: 소개 (Intro)
**제목**: Mind Palette - 창의적 사고 정리 플랫폼
**부제목**: AI와 함께하는 직관적인 아이디어 관리 시스템

**핵심 메시지**:
- 7일간의 집중 개발 프로젝트
- 5명의 전문가 팀 (PM, 디자이너, 프론트엔드, AI/데이터, 발표)
- Cursor AI 에이전트와의 협업으로 개발

**시각적 요소**:
- 프로젝트 로고/아이콘
- 팀원 소개 (역할별)
- 개발 기간 및 목표

---

## 슬라이드 2: 문제점 (Problem)
**제목**: 창의적 사고의 혼재와 정리 어려움

**현재 문제점**:
- 아이디어가 여러 곳에 흩어져 있음
- 사고 과정의 체계적 정리 부족
- 창의적 영감의 손실
- 협업 시 아이디어 공유의 어려움

**타겟 사용자**:
- 창의적 업무를 하는 전문가
- 팀 프로젝트를 진행하는 개발자
- 아이디어를 체계적으로 관리하고 싶은 사용자

**시각적 요소**:
- 혼재된 아이디어들을 보여주는 다이어그램
- 사용자 페인 포인트 일러스트레이션

---

## 슬라이드 3: 솔루션 (Solution)
**제목**: Mind Palette - AI 기반 사고 정리 솔루션

**핵심 기능**:
- AI 에이전트와의 실시간 협업
- 직관적인 시각적 인터페이스
- 자동 분류 및 태깅 시스템
- 팀 협업 기능

**기술적 특징**:
- Cursor AI 에이전트 통합
- Playwright 자동화 테스트
- Git 기반 협업 워크플로우
- 반응형 웹 디자인

**차별화 포인트**:
- AI와 인간의 창의적 협업
- 실시간 팀 협업 환경
- 체계적인 프로젝트 관리

**시각적 요소**:
- 플랫폼 UI/UX 디자인
- AI 에이전트 상호작용 시연
- 기술 아키텍처 다이어그램

---

## 슬라이드 4: 데모 (Demo)
**제목**: Mind Palette 실제 사용 시연

**데모 시나리오**:
1. **아이디어 입력**: 사용자가 창의적 아이디어 입력
2. **AI 분석**: AI 에이전트가 아이디어 분석 및 분류
3. **시각화**: 아이디어를 시각적으로 정리하여 표시
4. **협업**: 팀원들과 실시간 아이디어 공유
5. **관리**: 프로젝트별 아이디어 체계적 관리

**기술 데모**:
- Cursor AI 에이전트와의 대화
- Playwright 테스트 실행
- Git 협업 워크플로우 시연

**시각적 요소**:
- 실제 화면 녹화 영상
- 사용자 인터랙션 플로우
- AI 응답 및 분석 결과
