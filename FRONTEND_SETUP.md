# Mind Palette - 컬러 일기 앱 (1st Draft)

## 🎨 프로젝트 개요

감정을 색상으로 표현하고 기록하는 창의적인 일기 애플리케이션입니다.

## 🚀 기술 스택

- **React 18.2** - UI 라이브러리
- **Vite 5.0** - 빌드 도구 및 개발 서버
- **Lucide React** - 아이콘 라이브러리
- **CSS3** - 커스텀 유틸리티 클래스

## 📦 설치 및 실행

### 1. 의존성 설치

```bash
npm install
```

### 2. 개발 서버 실행

```bash
npm run dev
```

서버가 실행되면 자동으로 브라우저가 열리고 `http://localhost:3000`에서 앱을 확인할 수 있습니다.

### 3. 프로덕션 빌드

```bash
npm run build
```

### 4. 프로덕션 미리보기

```bash
npm run preview
```

## ✨ 주요 기능

### 1. 컬러 선택 (33색 팔레트)

- 11개씩 3줄로 구성된 컬러 팔레트
- 연한 색상, 중간 색상, 진한 색상으로 분류

### 2. 감정 선택

- 21가지 기본 감정 옵션
- '기타' 옵션으로 커스텀 감정 입력 가능

### 3. 에피소드 작성

- 자유로운 텍스트 입력
- 오늘 있었던 일이나 생각 기록

### 4. 시간대 선택

- 8개의 시간대 옵션 (새벽, 아침, 오전, 점심, 오후, 저녁, 밤, 심야)

### 5. 날씨 선택

- 9가지 날씨 옵션 (맑음, 흐림, 비, 눈, 바람, 안개, 천둥번개, 무더위, 추위)
- 각 날씨에 따른 맞춤형 느낌 옵션

### 6. 캘린더 뷰

- 월별 캘린더로 일기 확인
- 하루에 최대 4개의 일기 작성 가능
- 여러 일기가 있는 날은 색상이 혼합되어 표시
- 날짜 클릭으로 해당 일의 일기 상세 보기

## 🎯 사용 방법

1. **컬러 선택**: 지금 가장 끌리는 색상을 선택하세요
2. **감정 선택**: 현재 느끼는 감정을 선택하세요
3. **에피소드 작성**: 떠오르는 생각이나 오늘의 일을 적어보세요
4. **시간대 선택**: 지금은 언제인가요?
5. **날씨 선택**: 현재 날씨를 선택하세요
6. **날씨 느낌**: 이 날씨에 대한 느낌을 선택하고 저장하세요

## 🎨 디자인 시스템

### 색상 팔레트

- 연한 색상 (11개): 파스텔 톤
- 중간 색상 (11개): 비비드 톤
- 진한 색상 (11개): 딥 톤

- 폰트: Nanum Gothic, Apple SD Gothic Neo, Malgun Gothic

### 레이아웃

- 최대 너비: 672px (max-w-2xl)
