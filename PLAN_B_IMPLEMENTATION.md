# 플랜B (Figma 디자인 기반) 구현 내용

## 📋 개요

이 문서는 `feature/eunyoung-figma-test` 브랜치에서 구현된 플랜B의 주요 기능과 특징을 설명합니다.

## 🎨 디자인 기반

- **Figma 프로토타입**: https://image-human-51414311.figma.site/
- **디자인 컨셉**: 모바일 앱 스타일의 깔끔하고 직관적인 UI
- **색상 테마**: 파스텔 그라데이션 배경 (보라→하늘색)

---

## 🏠 1. 홈 화면 (Home Screen)

### 주요 기능

- ✅ **연속 기록 일수 표시**: 최근 연속으로 일기를 기록한 일수 (30 Days 원형 카드)
- ✅ **주간 요약**: 최근 7일간의 기록 여부를 색상 원형으로 표시
- ✅ **격려 메시지**: 기록을 계속할 때 격려 문구 표시
- ✅ **월별 캘린더**: 날짜별로 색상 원형으로 감정 기록 시각화
- ✅ **월 네비게이션**: 이전/다음 달 이동 기능

### UI 특징

- 파스텔 그라데이션 배경
- "Mind Palette" 필기체 제목
- 캘린더는 반투명 카드 형태 (blur 효과)
- 기록이 있는 날짜는 해당 일기의 첫 번째 색상으로 원형 표시

---

## ✍️ 2. 일기 쓰기 화면 (Diary Write Screen)

### 주요 기능

- ✅ **2페이지 구조**: "오늘의 일기 (1/2)", "오늘의 일기 (2/2)"
- ✅ **페이지 1**: 색상 선택
  - 끌리는 색 하나 선택 (필수)
  - 덜어내고 싶은 색 하나 선택 (선택)
- ✅ **페이지 2**: 감정 및 에피소드
  - 감정 선택 (12개 옵션 + 기타)
  - 에피소드 기록 (자유 텍스트)
- ✅ **저장 기능**: LocalStorage에 자동 저장

### UI 특징

- 모달 형태의 반투명 카드
- 색상은 33개 컬러 팔레트에서 선택
- 선택된 색상은 강조 표시 (그림자, 링 효과)
- 페이지 전환 인디케이터 표시

---

## 📊 3. 감정 보기 화면 (Emotion View Screen)

### 주요 기능

#### 3.1 통계 카드 (2개)

- ✅ **총 기록**: 기록한 총 일수 (중복 날짜 제거)
- ✅ **최다 감정**: 가장 많이 느낀 감정

#### 3.2 감정 컬러 팔레트

- ✅ 상위 5개 색상 표시
- ✅ 각 색상의 사용 횟수 및 퍼센트
- ✅ 가로 막대 그래프로 색상 비율 시각화
- ✅ 공유 버튼

#### 3.3 자주 느낀 감정

- ✅ 감정별 기록 횟수
- ✅ 각 감정을 기록했을 때 사용한 모든 색상 목록
- ✅ 최소 2회 이상 기록된 감정만 표시

#### 3.4 주간 기록 추이

- ✅ 최근 7주간의 기록 일수 표시
- ✅ 각 주별로 기록한 날짜 수

### UI 특징

- 섹션별로 카드 형태로 구분
- 통계 카드는 각각 다른 그라데이션 배경
- 색상은 원형으로 표시되고, 감정별로 색상 목록을 도트 형태로 표시

---

## 🔄 4. 네비게이션 시스템

### 하단 네비게이션 바

- ✅ **홈**: 홈 화면으로 이동
- ✅ **일기쓰기**: 일기 작성 화면으로 이동
- ✅ **감정 보기**: 감정 분석 화면으로 이동
- ✅ 활성 탭 하이라이트 표시

### 특징

- 고정 위치 (화면 하단)
- 반투명 배경 (blur 효과)
- 아이콘 + 텍스트 라벨

---

## 💾 5. 데이터 관리

### 저장 방식

- **LocalStorage**: 브라우저 로컬 저장소에 자동 저장
- **키**: `mind-palette-data`
- **형식**: JSON 배열

### 데이터 구조

```javascript
{
  id: string,
  color: string (HEX),
  avoidColor: string (HEX, 선택),
  emotion: string,
  episode: string,
  date: string (YYYY-MM-DD),
  timestamp: string,
  ...
}
```

### 실시간 업데이트

- 데이터 저장 시 자동으로 화면 갱신
- 다른 탭에서 변경사항도 감지하여 자동 동기화

---

## 🎨 6. 디자인 시스템

### 색상 팔레트

- **33개 색상**: 연한/중간/진한 톤으로 구성
- 각 색상은 HEX 코드로 저장 및 표시

### 스타일 특징

- **배경**: 파스텔 그라데이션 (#e0d2f7 → #a8d5f0)
- **카드**: 반투명 흰색 배경 + blur 효과
- **버튼**: 둥근 모서리, 부드러운 그림자
- **타이포그래피**: 나눔고딕 폰트 사용

---

## 📱 7. 반응형 디자인

- **최대 너비**: 428px (모바일 화면 기준)
- **중앙 정렬**: 데스크톱에서도 모바일 크기로 중앙 배치
- **스크롤 지원**: 내용이 많을 경우 세로 스크롤

---

## 🔧 8. 기술 스택

- **React 18**: UI 프레임워크
- **Vite**: 개발 환경 및 빌드 도구
- **Lucide React**: 아이콘 라이브러리
- **CSS**: 커스텀 스타일 (Tailwind 유틸리티 클래스 스타일)
