# 🎨 Mind Palette

**AI와 함께하는 창의적 사고 정리 플랫폼**

Mind Palette는 AI 에이전트와 인간의 창의적 협업을 통해 아이디어를 체계적으로 정리하고 관리할 수 있는 직관적인 플랫폼입니다. 7일간의 집중 개발을 통해 MVP를 완성하고, AI와 인간의 협업 모델을 제시합니다.

## 🎯 프로젝트 목적

### 핵심 목표

- **창의적 사고의 체계화**: 흩어진 아이디어들을 체계적으로 정리
- **AI-인간 협업 모델**: Cursor AI 에이전트와의 실시간 협업 구현
- **팀 협업 효율성**: 실시간 팀 협업 환경 구축
- **사용자 경험 최적화**: 직관적이고 사용하기 쉬운 인터페이스 제공

### 해결하고자 하는 문제

- 창의적 아이디어가 여러 곳에 흩어져 있어 정리하기 어려움
- 사고 과정의 체계적 정리 부족으로 인한 아이디어 손실
- 팀 협업 시 아이디어 공유의 비효율성
- 창의적 영감을 체계적으로 관리할 수 있는 도구 부족

## ✨ 주요 기능

### 🤖 AI 에이전트 협업

- **Cursor AI 에이전트 통합**: 실시간 AI와의 대화형 협업
- **역할별 AI 에이전트**: PM, 디자이너, 개발자, AI/데이터, 발표 담당자별 전문화된 AI
- **자동 분석**: AI가 아이디어를 자동으로 분석하고 분류
- **지능형 제안**: AI가 창의적 아이디어 발전 방향 제안

### 🎯 직관적 인터페이스

- **시각적 아이디어 정리**: 아이디어를 시각적으로 표현하고 연결
- **드래그 앤 드롭**: 직관적인 아이디어 재배치 및 정리
- **태그 시스템**: AI 기반 자동 태깅 및 수동 태그 관리
- **검색 및 필터링**: 강력한 검색 기능으로 아이디어 빠른 찾기

### 🔄 자동 분류 및 관리

- **AI 기반 분류**: 머신러닝을 통한 아이디어 자동 분류
- **카테고리 관리**: 프로젝트별, 주제별 아이디어 체계적 관리
- **우선순위 설정**: 중요도에 따른 아이디어 우선순위 자동 설정
- **연관성 분석**: 아이디어 간의 연관성 자동 분석 및 시각화

### 👥 팀 협업 기능

- **실시간 협업**: 팀원들과 실시간으로 아이디어 공유 및 편집
- **버전 관리**: Git 기반 아이디어 버전 관리 및 히스토리 추적
- **코멘트 시스템**: 아이디어에 대한 피드백 및 토론
- **권한 관리**: 역할별 접근 권한 및 편집 권한 설정

### 📱 반응형 디자인

- **모바일 우선**: 모바일 기기에서 최적화된 사용자 경험
- **크로스 플랫폼**: 데스크톱, 태블릿, 모바일 모든 기기 지원
- **오프라인 지원**: 네트워크 연결 없이도 기본 기능 사용 가능
- **접근성**: 웹 접근성 가이드라인 준수

### Collaboration & Development

## 🛠️ 설치 및 실행

### 필수 요구사항

- Node.js 18.0.0 이상
- npm 8.0.0 이상
- Git

### 설치 방법

```bash
# 저장소 클론
git clone https://github.com/SeeYounsunny/Mind-Palette.git

# 프로젝트 디렉토리로 이동
cd Mind-Palette

# 의존성 설치
npm install

# 개발 서버 실행
npm run dev
```

```bash

npm test

# UI 모드로 테스트 실행
npm run test:ui
```

## 🎯 사용법

### 초기 설정

1. **프로젝트 클론**

   ```bash
   git clone https://github.com/SeeYounsunny/Mind-Palette.git
   cd Mind-Palette
   npm install
   ```

2. **AI 에이전트 설정**
   - Cursor에서 AI 에이전트 활성화
   - 역할별 프롬프트 설정 (PM, 디자이너, 개발자, AI/데이터, 발표)
   - 협업 환경 구성

### 기본 사용법

#### 아이디어 입력 및 관리

1. **아이디어 입력**: 텍스트, 이미지, 링크 등 다양한 형태로 아이디어 입력
2. **AI 분석**: AI 에이전트가 자동으로 아이디어 분석 및 분류
3. **시각적 정리**: 분석 결과를 시각적으로 표현하여 이해하기 쉽게 정리
4. **태그 관리**: AI 기반 자동 태깅 및 수동 태그 추가/수정

#### 팀 협업

1. **실시간 공유**: 팀원들과 실시간으로 아이디어 공유
2. **피드백 시스템**: 아이디어에 대한 코멘트 및 피드백 작성
3. **버전 관리**: Git을 통한 아이디어 버전 관리 및 히스토리 추적
4. **권한 관리**: 역할별 접근 권한 및 편집 권한 설정

#### 고급 기능

1. **연관성 분석**: AI가 아이디어 간의 연관성 분석 및 시각화
2. **우선순위 설정**: 중요도에 따른 아이디어 우선순위 자동 설정
3. **검색 및 필터링**: 강력한 검색 기능으로 원하는 아이디어 빠른 찾기
4. **내보내기**: 아이디어를 다양한 형태로 내보내기 (PDF, 이미지, 텍스트)

### AI 에이전트 활용법

#### 역할별 AI 에이전트 사용

- **PM 에이전트**: 프로젝트 관리, 일정 조율, 문서화
- **디자이너 에이전트**: UI/UX 디자인, 사용자 경험 최적화
- **개발자 에이전트**: 코드 작성, 기술적 구현, 성능 최적화
- **AI/데이터 에이전트**: 데이터 분석, 머신러닝 모델 구축
- **발표 에이전트**: 발표자료 제작, 데모 준비, 프레젠테이션

#### AI 협업 팁

1. **명확한 지시**: AI에게 구체적이고 명확한 작업 지시
2. **피드백 제공**: AI의 결과에 대한 피드백 제공으로 성능 향상
3. **반복 개선**: AI와의 대화를 통한 지속적인 개선
4. **역할 분담**: 각 AI 에이전트의 전문성을 활용한 효율적 협업
